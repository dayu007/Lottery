# Lottery
基于H5的抽奖程序
开源一款基于H5的抽奖程序，能加载xls文件（需要转为utf-8编码）。
可以设定一等奖，二等奖，三等奖的数量及奖品
原则上修改代码后可任意设定奖项。
利用了undersore.js的shuffle函数对候选数组进行洗牌的方式做随机。
手机号码中间四位做了隐藏，利用了正则表达式的方式。
